Wykonaj następujące kroki, aby zaimportować lub przeprowadzić migrację poczty e-mail, kontaktów i kalendarza z gmaila do programu Outlook przy użyciu platformy Microsoft 365:
  
- [Importowanie gmaila do programu Outlook](https://support.microsoft.com/office/20fdb8f2-fed8-4b14-baf0-bf04b9c44bf7)
    
- [Importowanie kontaktów do programu Outlook](https://support.microsoft.com/office/bb796340-b58a-46c1-90c7-b549b8f3c5f8)
    
- [Importowanie kalendarza Google](https://support.microsoft.com/office/098ed60c-936b-41fb-83d6-7e3786437330)

## <a name="import-outlook-pst-files-to-microsoft-365-desktop"></a>Importowanie plików pst programu Outlook na platformę Microsoft 365 (pulpit)

Wykonaj następujące kroki, aby wyeksportować wiadomości e-mail, kontakty i kalendarze z programu Outlook do pliku pst, a następnie zaimportować ten plik do programu Outlook przy użyciu platformy Microsoft 365:
  
1. [Eksportowanie poczty e-mail, kontaktów i kalendarza](https://support.microsoft.com/office/14252b52-3075-4e9b-be4e-ff9ef1068f91)
    
2. [Importowanie poczty, kontaktów i kalendarza](https://support.microsoft.com/office/431a8e9a-f99f-4d5f-ae48-ded54b3440ac)
    
Jeśli chcesz tylko kontaktów, wykonaj następujące kroki:
  
1. [Eksportowanie kontaktów z programu Outlook](https://support.microsoft.com/office/10f09abd-643c-4495-bb80-543714eca73f)
    
2. [Importowanie kontaktów do programu Outlook](https://support.microsoft.com/office/bb796340-b58a-46c1-90c7-b549b8f3c5f8)
    
Aby rozpocząć proces, otwórz program Outlook i wybierz pozycję **Otwórz &amp;** **plik** \> Eksportuj \> **import/eksport**.

## <a name="see-other-email-accounts-in-outlook"></a>Zobacz inne konta e-mail w programie Outlook

Po prostu chcesz zobaczyć swój adres e-mail od innego dostawcy (takiego jak Gmail, Yahoo lub Live.com) w programie Outlook? Nie trzeba importować ani migrować. Możesz skonfigurować program Outlook lub Outlook Web App, aby można było uzyskiwać dostęp do innych kont z tego samego miejsca co skrzynka pocztowa platformy Microsoft 365 oraz wysyłać, odbierać i odczytywać wiadomości e-mail wysyłane do tych kont.
  
### <a name="outlook-desktop"></a>Outlook (pulpit)

Dodaj konto, takie jak prywatne konto Gmail, do programu Outlook.
  
- Otwórz program Outlook, a następnie przejdź do pozycji **Dodaj plik** \> **konta**.
    
Potrzebujesz bardziej szczegółowych kroków? Zobacz [Dodawanie konta](https://support.microsoft.com/office/6e27792a-9267-4aa4-8bb6-c84ef146101b).

## <a name="multiple-mailboxes-admins-can-bulk-import-email-contacts-and-calendars-to-microsoft-365"></a>Wiele skrzynek pocztowych: Administratorzy mogą zbiorczo importować wiadomości e-mail, kontakty i kalendarze do platformy Microsoft 365

W zależności od źródłowego systemu poczty e-mail można wybrać jedną z kilku metod migracji zbiorczej. Przeczytaj [artykuł Sposoby migracji wielu kont e-mail na platformę Microsoft 365](/Exchange/mailbox-migration/mailbox-migration) , aby zdecydować, która metoda działa.
